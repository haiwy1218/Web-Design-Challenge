Weather Layout

The purpose of this project was to analyze weather data and its changes as you get closer to the equator. Analyzing this data we used OpenWeatherMap API which gave a 500 city data set. MatPlotLib was used to plot the various comparisons with Latitude. The comparisons that were looked at were Max Temp, Cloudiness, Humidity and Wind Speed. This website includes visualiztions, data sets and descriptions of each plot.
